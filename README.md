### Player Controls
| Key | Action |
| ------ | ------ |
| W | Forward |
| A | Left |
| S | Backward |
| D | Right |
| Mouse | Move to Pan Camera |
| P | Open Pause Menu |
| U | Open Ability Upgrades Menu |

> The above controls are only for the overworld. In combat, the player is can only click on the UI displayed.
